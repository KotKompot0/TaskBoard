TaskBoard — это backend проект для управления задачами, похожий на GitHub issues, построенный на Go. 

## 🚀 Возможности

- 🔒 Аутентификация и авторизация пользователей
- 📝 Операции CRUD для задач
- 👥 Назначение задач и отслеживание статуса
- 🌐 RESTful API
- 🐳 Поддержка Docker

## 🛠️ Технологии

- **Go**
- **Gin**
- **GORM**
- **Docker**

## 🚀 Начало работы

### 📋 Предварительные требования

- Go 1.16+
- Docker

### 🛠️ Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Permyakov-Dmitriy/TaskBoard.git
   cd TaskBoard
   ```

2. Создайте файл .env и настройте переменные окружения;

3. Соберите и запустите:
   ```bash
   docker compose up --build
   ```
